# 介绍

在本路径之下，实现了fisco-bcos的异构链接入插件，包括链下插件及链上插件部分

- **offchain-plugin**：链下插件，使用maven管理的Java工程，使用maven编译即可。基于fisco`2.8.0`版本的javasdk开发，在fisco`2.0`国密链上测试通过。
